Overview

RoLocate is a free userscript that focuses on adding new features to Roblox in a way that feels as if they were built into the platform from the start. Its most popular feature is the server filtering features, but it includes plenty of other helpful features like SmartSearch.

Note: All data processing is done locally in your browser.

Key Benefits

  • Local Execution: Everything is done locally in your browser.
  • Open Source: Source code is available on greasyfork!

Getting Started

After installing the userscript it will start working on Roblox.com

First Launch

The first time you go on to roblox.com with the userscript a popup will appear telling you about the recent updates. You can close this.

Quick Start: All settings can be changed on the settings menu on the top right corner.

Interface Overview

Here are where the RoLocate buttons are located:

  • Settings Menu: Top right corner. Looks like the RoLocate logo.
  • Filter Button: The Filter button is available on games under the server tab. It will be right next to "Exclude Full Servers."
  • Recent Servers: Will be located on the server tab on games and under the private server list.
  • SmartSearch: Replaces the default Roblox search bar at the top of every page.

Server Filters

RoLocate allows you to filter through servers to find exactly what you're looking for.

  • Smallest Servers: Shows emptiest servers.
  • Available Space: Shows servers with space.
  • Player Count: Find servers with a specific player count.
  • Server Region: Shows a variety of server locations to join.
  • Best Connection: Automatically join the fastest server.
  • Newest Server: Find servers that are less than 5 minutes old.

How to Use Server Filters:

  1. Navigate to any Roblox game page
  2. Click on the "Servers" tab
  3. Look for the "Filter" button next to "Exclude Full Servers"
  4. Click the Filter button to open the filtering options
  5. And enjoy! :)

SmartSearch

Lightning fast search that shows results as you type with almost instant results.

Key Advantages:

  • Instant Results: See games, users, groups, and items appear as you type
  • Intelligent Sorting: Most relevant results appear first based on similarity
  • Tab-Based Navigation: Switch between Games, Users, Groups, and Items

How to Use SmartSearch:

  1. Click on the SmartSearch bar at the top of the page
  2. Start typing
  3. Instantly see results appear in the dropdown
  4. Switch between Games, Users, Groups, and Items using the tabs
  5. Click any result to go directly to that page

Performance Note: SmartSearch loads cards immediately while fetching thumbnails in the background for maximum responsiveness. This can result in slowdowns on very old devices.

Remove All Roblox Ads

Enjoy Roblox without ads!

How to Enable:

  1. Click the RoLocate settings button (top right corner)
  2. Click the Appearance tab
  3. Toggle "Remove All Roblox Ads"

What Gets Removed:

  • All banner ads
  • Sponsored game ads
  • Today's Pick on the homepage
  • Recommended section on the homepage

Recent Servers

Quickly return to the server your were previously in.

Features:

  • Last 3 Days: Automatically tracks your recent servers for the last 3 days
  • One Click Return: Rejoin servers with a single click
  • Server Details: Shows player count and region information

How to Access Recent Servers:

  1. Go to any Roblox game page
  2. Click on the "Servers" tab
  3. Look for the "Recent Servers" section below private servers
  4. Click any server to rejoin instantly

Game Quality Filter

Filter out low-quality games based on ratings and player counts.

Filter Criteria:

  • Minimum Rating: Hide games below a certain rating threshold
  • Player Count Range: Filter games by active player count
  • Content Quality: Detect and filter low-effort games
  • Popularity Threshold: Filter out obscure games with few players

How to Configure:

  1. Click the RoLocate settings button (top right corner)
  2. Click the Extras tab
  3. Toggle "Game Quality Filter"
  4. Click the "Edit" button
  5. Configure settings
  6. Click save and refresh the page

Mutual Friends

See if someone shares the same friend as you.

Features:

  • Friend Indicator: Shows which friends you share with others
  • Smooth Animtations: Looks like it was made by Roblox

How to Use:

  1. Go to any user's profile page
  2. And you can see the mutual friends!

Quick Launch Games

Quickly play your games on the homepage!

How to Configure:

  • Click on the Add Game icon on the homepage
  • Enter the Game ID and click Add Game
  • Click on your game to play directly from the homepage

Note: Sometimes it may not work due an error with the Roblox API. If this happens, try refreshing the page or waiting a few seconds. There is a maximum of 10 games that can be added to the quick launch list.

Troubleshooting

Common Error codes and issues

Filtering Error: Failed to obtain permission to send API requests to the Roblox API. Please allow the script to enable request sending.
  • 1. Reinstall the userscript manager.
  • 2. Install RoLocate.
  • 3. Go to any roblox game's page.
  • 4. Click on server tab then filter button.
  • 5. Click on server region.
  • 6. Proceed until 2 Tampermonkey/Violentmonkey tabs open up.
  • 7. Click "Always Allow" on both of them.
  • The issue should now be fixed.
Error: No servers found. Please try again later.
  • This usually means that the game is so popular that before RoLocate could scan the servers, the servers fill up to the max player count.
  • Try again later or try a different game.
  • If issue still persists then report on Greasyfork.

General Troubleshooting Steps:

If RoLocate isn't working:

  1. Refresh the Roblox page
  2. Check if your userscript manager is enabled
  3. Verify RoLocate is installed and enabled
  4. Disable other Roblox extensions temporarily
  5. Check browser console for errors (F12)
  6. Reinstall RoLocate if issues persist

Enable Console Logs: You can enable console logs by going to the settings menu and enabling "Enable Console Logs". This will help you debug issues.
